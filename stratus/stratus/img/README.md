# Icon asset

`icon-stratus.png` is the official Stratus Red Team badge
(https://github.com/DataDog/stratus-red-team, `docs/logo.png`) flattened onto a
solid opaque background, per the injector icon standard
(OpenAEV-Platform/injectors#305):

- Square 1:1, 512x512 PNG
- Solid opaque background (no transparency)
- Genuine brand artwork only (never invented / upscaled from low-res)
